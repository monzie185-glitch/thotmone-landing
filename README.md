# Thotm One – Landing (Halo Edition)

Landing page sombre premium avec halo vert, CTA email, animations subtiles, et balises Open Graph pour partage réseaux sociaux.

## Fichiers
- `index.html` – page principale
- `styles.css` – styles (halo, dark theme, CTA hover, fade-in)
- `script.js` – apparitions au scroll
- `favicon.svg` – icône
- `logo-thotm-one.jpg` – **à ajouter** (voir ci-dessous)
- `og-image.png` – (optionnel) image d’aperçu réseaux

## Logo
Ajoutez votre logo sous le nom **`logo-thotm-one.jpg`** (même dossier que `index.html`).

## Déploiement Vercel
1. Sur Vercel → **Add New Project** → **Import Git Repository** → sélectionnez `thotmone-landing`.
2. Framework preset: **Other** → Build Command: *(vide)* → Output: **/** (racine).
3. **Deploy** → vous obtenez une URL publique (`*.vercel.app`).

Bouton CTA → `mailto:contact@thotmone.com`.
